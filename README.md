# sxd-ui

## Install
```
npm i sxd-ui -S
```

### Import to your projects
```
import Vue from 'vue'
import SxdUi from 'sxd-ui'
import "sxd-ui/lib/sxd-ui.css"
import App from './App.vue'

Vue.use(SxdUi)

new Vue({
    el: '#app',
    render: h => h(App)
})
```
